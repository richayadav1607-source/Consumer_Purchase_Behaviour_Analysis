Analyzed consumer purchasing patterns using Python, SQL, and data visualization techniques to identify key factors influencing buying decisions. The project involved data cleaning, exploratory data analysis (EDA), customer segmentation, and trend analysis to uncover insights related to customer demographics, product preferences, spending habits, and purchase frequency. Interactive visualizations and dashboards were created to support data-driven business decisions and improve customer targeting strategies.
Key Skills Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL
Data Cleaning & Preprocessing
Data Visualization
Consumer Insights & Trend Analysis
Dashboard Creation (Power BI)
